# 非环无机取代基化合物命名工具

一个用于生成非环无机取代基化合物（包括卤代化合物、硝基化合物、氨基化合物等）系统命名的小工具。

## 项目简介

这是一个实用的化学工具程序，由编程初学者逐步完善而成。它实现了有机化学中含无机取代基的非环化合物基本命名规则，支持带支链和多重取代基结构的系统命名。

## 主要作用

该程序的核心功能是：**根据用户输入的碳骨架结构（包括支链和无机取代基信息），自动输出对应的无机取代基化合物系统名称**。

具体来说，它可以：

1.  **识别卤代烃**：输入碳数、卤素种类（氟、氯、溴、碘）、位置和支链信息，输出如"氯甲烷"、"2-氯丁烷"、"1-溴-2-甲基丙烷"、"2,3-二氯-1-碘丁烷"等名称。根据IUPAC规则，多种卤素共存时按氟、氯、溴、碘顺序排列。
2.  **识别硝基化合物**：输入碳数、硝基(-NO₂)位置和支链信息，输出如"硝基甲烷"、"1-硝基丙烷"、"2-甲基-1-硝基丁烷"等名称。
3.  **识别氨基化合物**：输入碳数、氨基(-NH₂)位置和支链信息，输出如"甲胺"、"丙-1-胺"、"3-氨基-2-甲基戊烷"等名称。
4.  **识别混合取代基化合物**：输入含多种无机取代基（如同时含卤素和硝基）的结构，输出如"1-氯-2-硝基丙烷"、"3-氨基-1-溴丁烷"等名称。不同取代基按优先级倒序排列。
5.  **支持导入导出**：可通过文件或代码两种方式，方便地与你的朋友分享化合物。

## 程序特点

- **规则化**：程序内部嵌入了国际纯粹与应用化学联合会（IUPAC）的非环无机取代基化合物命名规则。
- **优先级处理**：正确处理取代基优先级——当氨基与羟基共存时，羟基优先作为主官能团；卤素和硝基始终作为前缀取代基。
- **简单直接**：用户交互清晰，输入必要信息即可获得标准化的命名结果。
- **学习工具**：对于化学学习者，它可以作为验证命名规则理解和快速查阅的辅助工具。
